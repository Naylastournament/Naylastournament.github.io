Rules and settings

If a player drop and the game cannot be resumed we will try to view the recorded game.
If one player was going to win without any doubt he/she will be awarded the win.
If there was no obvious winner then the game will be replayed with the exact same settings.
If the same player constantly disconnects, he/she will forfeit and the other player will progress
to the next round.

If a player is causing unplayable lag and can clearly be proven as the
sole cause of the lag then that player will be disqualified.
